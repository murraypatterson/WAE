# WAE

A lexer and parser for the WAE (with arithmetic expressions) language

## Run it yourself

[![Run on Repl.it](https://repl.it/badge/github/murraypatterson/WAE)](https://repl.it/github/murraypatterson/WAE)

*Note: that repl.it can seem to hang on the "detecting language" part,
but just click "reload" on your browser, and it seems to resolve that
problem, click "Run" and enjoy!
